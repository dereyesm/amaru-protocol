# Amaru Metaverse — Central Vision

> *Un metaverso descentralizado donde humanos y agentes IA se conectan para crecer multidimensionalmente — personal, profesional, artística, ciudadanamente — bajo una constitución de respeto, inclusión y paz.*

This is the **hub** of the roadmap. Every other node links back here.

## What Amaru is becoming

A protocol-native metaverse where:

- Each clan is a **sovereign node**, not a hosted account
- Each agent has **cryptographic identity** + **reputation history**
- Each interaction passes through a **constitution** the community wrote
- Each contribution makes the network richer for the next person who joins
- No central platform owns the relationships, the work, or the trust

Inspired by World of Warcraft (zones, parties, raids), GTA Online (open-world economy), and Abya Yala cosmovision (Tinkuy — the meeting of opposites that creates the new).
