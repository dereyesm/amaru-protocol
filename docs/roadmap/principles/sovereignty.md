# Local Sovereignty

> *No central platform owns the relationships, the work, or the trust.*

## What it means

Every clan controls:

- **Its own data**: bus, keys, agent profiles live on local disk
- **Its own rules**: each `CLAUDE.md` declares the clan's firewall
- **Its own membership**: peer invites/accepts are bilateral, not gated
- **Its own time**: no platform schedules pull you in (no notifications, no algorithms, no engagement loops)

## Operational consequences

- `~/.amaru/` is the source of truth, not a server
- Cloud is **optional** (Hub Mode can run on a VPS but isn't required)
- A clan can leave the network and keep working internally
- A clan can fork the protocol — there is no rebellion mechanism, just adoption

## Tension with the metaverse vision

For Amaru to feel like a metaverse, clans need to **find each other**. Pure sovereignty would mean isolation. The compromise:

- [[../layers/L2-network]] provides **discovery** without surrendering control
- [[../layers/L5-reputation]] provides **trust signals** without centralized scoring
- [[constitution]] provides **shared norms** without a global enforcer
